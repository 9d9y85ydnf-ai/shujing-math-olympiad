# 中国高中数学竞赛代数解题规范

## 1. 成品目标

一份合格的竞赛解析应同时满足：

- 结论正确；
- 条件使用完整；
- 每一步推导可复核；
- 方法难度符合高中竞赛；
- 等号或最值取值说明完整；
- 读者能看出为什么选择这种方法。

默认成品先给一条最稳健的主解，只有在另一种解法体现重要方法时才补充多解。

## 2. 审题与预处理

### 2.1 条件清单

先写清楚变量是正数、非负数还是任意实数，并检查分母、根式和对数的定义域。

### 2.2 对称性

- 对称式可以交换变量，常考虑令两个变量相等或使用 (p,q,r)。
- 循环式只能循环置换，不能任意交换变量。
- 若题目含有排序条件，应先判断能否使用排序不等式或切比雪夫不等式。

### 2.3 齐次性与归一化

若不等式齐次，可令 (a+b+c=1)、(abc=1) 或其他方便的量为常数。归一化后必须说明原因，并保持等号条件可回代。

## 3. 常用方法

### 3.1 均值不等式与局部放缩

当目标中出现高次项、分式或需要线性化时，可以人为补入若干相同项，使均值不等式的等号条件与目标取值一致。

典型形式：

\[
x^m+(m-1)t\ge m\sqrt[m]{x^m t^{m-1}}.
\]

选择 (t) 的原则不是机械套公式，而是让右端变成目标中需要的线性项或固定量。使用后应立即说明 (t) 的取值和等号条件。

### 3.2 柯西不等式

分式和的常用入口是 Engel 形式：

\[
\sum_{i=1}^n\frac{x_i^2}{y_i}\ge
\frac{(x_1+x_2+\cdots+x_n)^2}{y_1+y_2+\cdots+y_n},
\qquad y_i>0.
\]

选择分子时，优先使分子和题设约束相同，选择分母时，优先使分母和能够继续用均值、排序或已知条件估计。

### 3.3 排序与切比雪夫

当两组量具有相同的单调顺序时，可比较同序和反序配对。使用前要明确排序关系，不能只凭表达式外观判断。

切比雪夫不等式适合处理两列同序数列的乘积和，尤其适用于先排序再把复杂项拆成两列的情形。

### 3.4 调整法与局部放缩

调整法的核心是：在保持关键约束不变的前提下，改变局部变量，证明目标值不会变小或不会变大，从而把问题推到边界、等值点或变量相等的情形。

标准步骤：

1. 找到需要保持不变的和、积或对称量；
2. 选取两个变量进行局部调整；
3. 比较调整前后的目标差；
4. 判断差值符号；
5. 重复调整直到得到可处理的边界情形；
6. 回查所有等号条件。

局部放缩则是把复杂表达式拆成若干个局部不等式，使每个局部等号都能服务于最终等号。

### 3.5 琴生不等式与切线法

在高中竞赛成品中，优先使用以下代数表达：

\[
f(x)\ge kx+b
\quad\text{或}\quad
f(x)\le kx+b.
\]

切线点通常由等号候选或对称点确定。得到候选直线后，应通过移项、通分、因式分解或二次判别证明其对整个定义域成立，而不是只写“由图像可知”。

### 3.6 三元对称式的 (pqr) 法

对于三元对称式，记

\[
p=x+y+z,\qquad q=xy+yz+zx,\qquad r=xyz.
\]

先把目标和条件化为 (p,q,r)，再使用基本关系，例如

\[
p^2\ge 3q,
\qquad q^2\ge 3pr,
\qquad p^3+9r\ge 4pq.
\]

含 (r) 的表达式常可视为关于 (r) 的一次式或低次式。此时要说明允许范围的边界，常见边界对应两个变量相等，而不能直接无条件写成“令 (y=z)”。

### 3.7 张瑞祥法与等号处理

讲义中的张瑞祥法可理解为：先锁定等号元组，再利用换序、配方、对称展开和差分，把目标拆成一组在该等号结构下显然非负的部分。

使用时应遵循：

1. 先从对称性和数值试探确定等号候选；
2. 把目标减去等号处的常数；
3. 按变量交换或分组寻找成对抵消；
4. 将剩余部分写成平方、乘积或已知非负量；
5. 最后完整回收等号条件。

该方法适合结构性较强的对称不等式，不适合没有清晰等号候选时的盲目展开。

## 4. 题解输出格式

正式题解建议采用：

```text
解：
先说明必要的变形或不妨设。
逐步证明目标不等式或求出最值。
说明等号成立条件，并验证其满足题设。

思路：
说明识别出的结构和方法入口。

点评：
说明方法可迁移的题型、边界或易错点。
```

“思路”和“点评”可以省略，但“解”与等号条件不能省略。

## 5. 最终检查清单

- [ ] 所有变量条件均已使用或说明不需要使用。
- [ ] 每个分母、根式和除法步骤合法。
- [ ] 所有不等式方向正确。
- [ ] 每次使用定理都能对应到本题具体表达式。
- [ ] 等号条件在每一步都兼容。
- [ ] 最终取值确实可以达到。
- [ ] 没有把循环式当成对称式。
- [ ] 没有把超出高中范围的工具伪装成初等结论。
