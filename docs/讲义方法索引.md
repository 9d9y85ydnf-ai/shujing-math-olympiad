# 讲义方法索引

来源：用户提供的《普通基本的古典不等式技术》讲义。讲义以不等式竞赛题为主，核心内容可归纳为四个方法模块，并辅以均值、柯西、排序、(pqr) 和等号分析。

## I. 调整法与局部放缩

### 核心思想

保持和、积或其他关键约束不变，对变量做局部调整，把极值推向变量相等、边界或少数变量非零的情形。

### 识别信号

- 题设给出固定和或固定积；
- 目标在交换两个变量后变化可控；
- 数值范围或幂次差异很大；
- 直接套均值不等式会产生不匹配的等号。

### 操作模板

1. 固定其余变量；
2. 用一个参数表示待调整变量；
3. 比较目标函数在调整前后的差；
4. 证明差值具有固定符号；
5. 迭代到边界或等值状态。

### 等号

等号通常来自变量相等、调整参数取边界，或所有局部放缩同时取等。必须把局部等号条件交集写出来。

## II. 琴生不等式与切线法

### 核心思想

构造一条在整个定义域成立的直线，对单项函数作统一的线性估计，再利用题设的和、积或对称条件求和。

### 识别信号

- 目标是 Σf(x_i)、∏ f(x_i) 或对称函数和；
- 等号候选为所有变量相等；
- 可以找到一个简单的切点或接触点；
- 直接使用均值不等式难以匹配目标常数。

### 高中竞赛写法

优先证明

\[
f(x)-kx-b\ge 0
\]

可以通过因式分解、配方、二次判别或基本不等式完成。导数只作为内部寻找候选切线的辅助，不作为默认成品证明。

## III. (pqr) 法

### 基本代换

\[
p=x+y+z,
\qquad q=xy+yz+zx,
\qquad r=xyz.
\]

### 常用关系

\[
p^2\ge 3q,
\qquad q^2\ge 3pr,
\qquad p^3+9r\ge 4pq.
\]

此外，

\[
x^2+y^2+z^2=p^2-2q,
\]

\[
x^3+y^3+z^3=p^3-3pq+3r.
\]

### 标准流程

1. 判断目标是否为三元对称式；
2. 化为 (p,q,r)；
3. 利用题设消去一个量；
4. 将剩余表达式看成一个变量的函数或多项式；
5. 处理变量允许范围和边界；
6. 检查边界是否对应两个变量相等或其他退化情形。

### 常见误用

不能因为题目是三元式就直接令 (y=z)。必须先说明对称性、降维依据或边界原理。

## IV. 张瑞祥法

### 核心思想

围绕等号元组进行对称展开、换序、配方和差分拆解，把目标写成若干个非负项之和。

### 适用条件

- 等号元组容易猜测；
- 目标具有明显对称性；
- 展开后能形成成对抵消或平方结构；
- 传统均值、柯西路线难以匹配精确常数。

### 操作模板

1. 先求或猜等号点；
2. 将目标减去等号值；
3. 进行有目的的换序和分组；
4. 提取平方、差平方或已知非负因子；
5. 从分解式反推完整等号条件。

### 注意

该方法的关键是“有结构地展开”，不是无目标地暴力展开。正式解答中必须解释分组依据。

## 基础工具索引

| 工具 | 主要入口 | 使用前检查 |
| --- | --- | --- |
| 均值不等式 | 高次项、乘积约束、线性化 | 各项非负，等号能否匹配 |
| 柯西不等式 | 分式和、平方和、积的下界 | 分母正性，分子选择是否服务于题设 |
| 排序不等式 | 两列同序或反序配对 | 是否真的有排序关系 |
| 切比雪夫不等式 | 同序数列乘积和 | 两列单调方向一致 |
| 归纳法 | 项数 (n) 变化、递推结构 | 基础步、归纳假设、递推步完整 |
| (pqr) | 三元对称式 | 是否对称，参数范围是否明确 |

## 方法选择顺序

先看定义域和齐次性，再看对称性与等号，最后选择最短且等号链最清楚的方法。方法索引不是固定算法，不能牺牲证明的透明度来追求技巧名称。
